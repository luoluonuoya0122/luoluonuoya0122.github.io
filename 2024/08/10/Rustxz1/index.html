运行代码

1、ructc main.rs

2、cargo run

3、cargo build

4、cargo build –realse 如果要正式发布的时候。我们会添加–release参数来编译

5、cargo check

当项目变大之后，无论是cargo run还是cargo build都会不可避免的变慢，cargo check这个命令则是一种更快的验证我们代码的编译正确性。它的作用很简单：快速的检查代码是否能被编译通过。这个命令速度非常快，因此能节省大量的编译时间：
